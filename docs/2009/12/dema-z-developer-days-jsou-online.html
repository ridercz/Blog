Dema z Developer Days jsou online

  •  Akce a události
Michal Altair Valášek  ​
2. prosince 2009 ​

Na webu MSTV.CZ jsou k dispozici záznamy z Microsoft Developer Days. Pokud se těšíte na záznam opiček, které jsme předváděli se Štěpánem Bechynským, tak vás ovšem musím zklamat, jsou to jenom zvlášť natočené screencasty.

Zveřejněny jsou všechny prezentace, pokud vás zajímají moje dema, tak zde jsou přímé odkazy:

  • HTML Code Snippets
  • ViewState v ASP.NET 4.0 (seriál o ViewState zde: 1 2 3 4)
  • URL routing
  • Rozšiřitelnost request validation
​ Je v článku něco špatně? Chcete něco doplnit?

Komentáře zde nenajdete, ale pokud je v článku chyba nebo k němu chcete něco věcného doplnit, můžete na GitHubu otevřít nový issue nebo navrhnout změnu v textu a poslat mi pull request.

Tamtéž také najdete historii všech verzí článku, pokud v něm byly provedeny nějaké změny či úpravy.
