Visual Studio 2013 RTW

  •  Software
Michal Altair Valášek  ​
17. října 2013 ​

Microsoft právě uvedl finální verzi Visual Studia 2013** a s ním související ASP.NET Web Tools** a .NET Framework 4.5.1**.

Podrobnější informace a možnost downloadu najdete na stránkách Visual Studia a na blogu ASP.NET týmu.

Hlavní novinky, které VS 2013 a .NET 4.5.1 přináší:

  • Browser Link** – propojení prohlížeče a Visual Studia. Výrazné usnadnění života pro webové vývojáře při ladění v několika browserech najednou.
  • One ASP.NET** – nový způsob vytváření a konfigurace webových projektů, umožňující snadno kombinovat jednotlivé technologie (Web Forms, MVC, WebAPI…) a formy autentizace.
  • ASP.NET Identity** – po osmi letech nový model autentizace a správy identit v ASP.NET aplikacích, nástupce membership provider modelu. Snadná federace identit třeba s Facebookem, Twitterem a dalšími službami.
  • Nové verze knihoven** – SignalR, Web API, Entity Framework a další se dočkaly updatů a změn.
  • OWIN** – Open Web Interface for .NET usnadňuje hostování ASP.NET aplikací mimo IIS – třeba ve vlastním procesu.

Pokud vás tyto novinky zajímají, zvu vás na své přednášky na konferenci MS Fest, a to 2. 11. v Praze nebo 30. 11. v Ostravě.

Můžete se podívat i na záznam mé přednášky z MS Festu v Brně, kde je ale vše předváděno na RC verzi, která neuměla některé pikošky :)
