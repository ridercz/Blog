Chcete na Windows spouštět aplikace pro Linux? Není to úplně běžný požadavek a pro typického uživatele to nedává smysl. Ale pro vývojáře a IT profesionály ano, protože existuje řada pro ně zajímavých nástrojů, které na Windows nefungují vůbec nebo jenom omezeně.

Nabízí se možnost použití virtuálního počítače, pomocí technologií jako Hyper-V, VirtualBox a další. Jejich problém je ale velký overhead a často špatná integrace s hostitelským operačním systémem.

## Windows Subsystem for Linux (WSL)

Od Windows 10 nabízí Microsoft Windows Subsystem for Linux. To je součást Windows, která umožňuje spouštět různé distribuce Linuxu (můžete jich mít na počítači několik současně) pod Windows. V úvodním videu série představuji WSL a ukazuji, jak nainstalovat vámi vybranou distribuci.

## Sdílení souborů mezi Windows a WSL

Windows a Linux používají různé souborové systémy a navzájem si moc nerozumí. Ale s WSL je to snadné. Můžete jednoduše z Linuxové části přistupovat k souborovému systému hostitelského stroje a naopak.

## Grafické aplikace pro Linux na Windows

Ačkoliv máme Linux spojený většinou s příkazovým řádkem, umožňuje spouštět i aplikace s grafickým uživatelským rozhraním. WSL je umí nejenom spouštět, ale i integrovat do systému velmi bezešvým způsobem.
